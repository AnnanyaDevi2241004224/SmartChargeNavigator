📖 About Smart EV Finder

🔍 What is this?

Smart EV Finder helps EV users locate charging stations using **Dijkstra’s Algorithm** to ensure the fastest route.

⚡ Key Features

🔹 **Shortest Route Calculation**
🔹 **Google Maps Integration**
🔹 **Multiple Providers Supported**
🔹 **Real-time Location Search**

🛠️ How It Works

Users select their EV brand, and the app finds nearby chargers using a **MySQL database**, calculates the shortest route, and displays it on **Google Maps**.

🚀 Future Enhancements

We aim to add **live availability tracking, user reviews, and integrated payments**.
